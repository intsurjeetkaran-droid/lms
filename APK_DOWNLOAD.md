### 📋 Installation Instructions

1. **Download the APK**
   - Click the download link above
   - Or navigate to the repository root and download `LaundryApp.apk`

2. **Transfer to Your Phone**
   - Via USB cable
   - Via email attachment
   - Via cloud storage (Google Drive, Dropbox, etc.)
   - Via direct download on phone browser

3. **Install on Android**
   - Open the APK file on your phone
   - Tap "Install" when prompted
   - If you see "Install from Unknown Sources" warning:
     - Go to Settings → Security
     - Enable "Unknown Sources" or "Install Unknown Apps"
     - Return and tap "Install" again

4. **Open the App**
   - Find "Laundry Service" icon on your home screen
   - Tap to open
   - Allow location permission when prompted
   - Start using the app!

### 🐛 Troubleshooting

**Installation Issues:**
- Make sure "Unknown Sources" is enabled
- Check if you have enough storage space
- Try redownloading the APK if corrupted

**Login Issues:**
- Check your internet connection
- Verify you're using correct credentials
- Make sure backend is accessible

**Location Issues:**
- Allow location permission when prompted
- Enable GPS in phone settings
- Check if location services are turned on

**App Crashes:**
- Clear app data in Settings → Apps
- Reinstall the app
- Check if your Android version is 5.1 or higher
